Sample Automation Script

Below is a sample of simple automation that can be employed for marketing or periodic report updates. The Python script prompts the user for the email's subject, body, recipient address, and sending interval. This script assumes you are using a Gmail account to send the email, so you'll need to enable "Less secure app access" in your Google account or use an App Password if Two-Factor Authentication is enabled. >

Before running this script: Replace your_email@gmail.com with your Gmail address. Replace your_password with your Gmail password or an App Password if you have Two-Factor Authentication enabled. Make sure you have the necessary permissions set in your Gmail account to allow less secure apps to send emails (if not using an App Password). This script will continue running and send emails at the specified intervals until you stop it manually (e.g., by pressing CTRL+C in the terminal). Note: Using your real password in scripts is not recommended for security reasons. It's better to use OAuth2 for Gmail or other secure methods in production code. Also, continuously sending emails at short intervals may violate your email provider's usage policies and could lead to your account being temporarily blocked or flagged for spam, so use this script responsibly.

import smtplib

from email.mime.text import MIMEText

from email.mime.multipart import MIMEMultipart

import schedule

import time

# User inputs

subject = input("Enter the email subject: ")

body = input("Enter the email body: ")

recipient_email = input("Enter the recipient's email address: ")

send_interval = int(input("Enter the sending interval in minutes: "))

# Email setup

sender_email = "your_email@gmail.com"

sender_password = "your_password"

def send_email():

# Create a multipart message

msg = MIMEMultipart()

msg['From'] = sender_email

msg['To'] = recipient_email

msg['Subject'] = subject

# Attach the email body

msg.attach(MIMEText(body, 'plain'))

# Server setup and send email

try:

server = smtplib.SMTP('smtp.gmail.com', 587)

server.starttls()

server.login(sender_email, sender_password)

text = msg.as_string()

server.sendmail(sender_email, recipient_email, text)

server.quit()

print(f"Email sent to {recipient_email}")

except Exception as e:

print(f"Failed to send email. Error: {e}")

# Schedule the email to be sent at intervals

schedule.every(send_interval).minutes.do(send_email)

print(f"Starting to send emails every {send_interval} minute(s)...")

try:

while True:

schedule.run_pending()

time.sleep(1)

except KeyboardInterrupt:

print("Stopped the email scheduler.")
